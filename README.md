# A-Galeria-de-Fotos-Simples-e-Responsiva
A chave é a propriedade grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)), que cria uma galeria fluida: o Grid ajusta o número de colunas automaticamente para que cada imagem tenha no mínimo 250px de largura, preenchendo o espaço restante de forma igual. O design inclui um efeito de zoom suave (transform: scale) no hover.
